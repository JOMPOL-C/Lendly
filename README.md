# LENDLY

An **online costume rental platform**

---

## Features

- Customer registration, shop registration, and identity verification
- Forgot password system
- Product filtering by category, body measurements, and size
- Availability checking via a rental calendar on the product page
- Return-date reminders
- Real-time chat between customers and shops
- Damage reporting by shops with penalty details via in-app chat (for rental cases)
- Shop management features:
  - View shop information
  - Add, edit, and delete products
  - Create promotions
  - Manage shop data
- Rental cancellation requests in case of issues  
  - (Requires agreement from both customer and shop)
- Issue reporting and request submission system
- Product reviews (no rating system)
- Shipment tracking
- Online payments via Omise
